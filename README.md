# WavePlayer

WavePlayer is a simple music streaming web app built with React.

## Features

- Play and pause songs
- Mini music player
- Highlight active song
- Next and previous controls
- Responsive UI
- Clean dark mode design

## Built With

- React
- CSS
- Vite

## How to Run

1. Clone the repository
2. Run npm install
3. Run npm run dev
